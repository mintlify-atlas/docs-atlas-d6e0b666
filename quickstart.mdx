## System requirements

Forsaken Casino runs entirely in the browser with no installation required. You need:

- A modern browser with WebGL support (Chrome, Firefox, Edge, or Safari — recent versions)
- No plugins, extensions, or package manager

<Note>
  Background music and sound effects require a browser that supports the Web Audio API. All modern browsers do. The audio does **not** start on page load — it begins on your first card hover. This is by design: the `startTimer()` function calls `playSound()` only when you interact with a card for the first time, working around browser autoplay restrictions.
</Note>

## Setup

<Steps>
  <Step title="Clone or download the repository">
    Clone the repository from GitHub:

    ```bash
    git clone https://github.com/Chernobyl25/memorama.git
    ```

    Or download the ZIP from the GitHub repository page and extract it to a local folder.
  </Step>

  <Step title="Confirm the file structure">
    The project is a single HTML file plus its assets. Your directory should look like this:

    ```
    memorama/
    ├── index.html
    ├── img/
    │   ├── img1.jpg
    │   ├── img2.jpg
    │   ├── img3.jpg
    │   ├── img4.jpg
    │   └── atras.jpeg
    ├── piso.jpg
    ├── lados.jpg
    ├── cartas.mp3
    └── fondo.mp3
    ```

    All image and audio assets must be present alongside `index.html` for the game to load correctly.
  </Step>

  <Step title="Open the game">
    You have two options:

    **Option A — Open directly (simplest):**
    Double-click `index.html` or drag it into your browser window.

    **Option B — Serve over HTTP (recommended for audio):**
    Some browsers block audio when files are loaded via the `file://` protocol. Serving locally avoids this:

    ```bash
    # Python 3
    python -m http.server 8000
    ```

    Then open `http://localhost:8000` in your browser.
  </Step>

  <Step title="Start playing">
    Once the page loads you will see the 3D casino scene. Hover your cursor over any card on the table to begin — this starts the 40-second **VIDA** countdown and triggers the background music.
  </Step>
</Steps>

## What to expect when you first open the game

When `index.html` loads:

- A dark 3D casino environment renders with exponential red fog, a textured floor and back wall, and a flickering red point light overhead.
- Eight face-down cards are arranged in a **4×2 grid** on the round table in front of you.
- The **VIDA: 40s** timer is visible in the top center of your view, but it is not running yet.
- No music plays until you hover over your first card.

The game begins the moment you hover over a card.

## VR headset vs. desktop play

Forsaken Casino is built with [A-Frame 1.4.0](https://aframe.io), which supports both desktop and WebVR/WebXR devices.

| Mode | How to play |
|---|---|
| Desktop | Use the mouse to look around and hover over cards. WASD keys move the camera. |
| VR headset | Open the page in a WebXR-compatible browser on your headset. A-Frame will detect the device and offer an immersive mode entry button. |

<Tip>
  For the best desktop experience, keep the default camera angle — the scene is positioned so the card table is fully visible from the starting position without needing to move.
</Tip>
